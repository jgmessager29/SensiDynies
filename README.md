===== SENSIDYNIES BOT — GUIDE UTILISATEUR FR =====

SensiDynies est un bot Discord dédié aux communautés francophones autour des maladies chroniques et du bien-être, conçu pour offrir un environnement sécurisé, respectueux et interactif.

A/ OBEJECTIFS PRINCIPAUX :

- Sécurité et modération : protection des utilisateurs grâce à la suppression automatique des contenus inappropriés, mots interdits, adresses email ou numéros de téléphone.
- Accessibilité à l’information : accès rapide à des définitions médicales fiables via !dico.
- Amusement et interaction : blagues aléatoires, choix aléatoires, réactions automatiques pour rendre les échanges plus conviviaux.
- Organisation du serveur : aide à la navigation et à la compréhension des différentes catégories avec !perdu.
- Communication simplifiée : informations sur le bot, règles du serveur et guide des commandes accessibles facilement.

B/ FONCTIONNALITES PRINCIPALES :

- Détection automatique des messages contenant des propos inappropriés ou des informations personnelles.
- Logs détaillés dans un canal de modération pour suivre les actions du bot.
- Commandes interactives pour aider les membres à trouver des informations, s’amuser ou s’orienter.
- Embeds stylisés pour rendre les messages plus lisibles et attractifs.
- Compatible avec les emojis personnalisés du serveur pour les réactions automatiques.

C/ COMMANDE DU BOT

1. !aide [commande] : Affiche toutes les commandes ou le détail d’une commande spécifique.
2. !dico [mot] : Fournit la définition d’un mot médical.
3. !blague : Poste une blague aléatoire pour détendre l’atmosphère.
4. !choix [option1, option2, ...] : Choisit aléatoirement une option parmi plusieurs proposées.
5. !effacer [nombre] : Supprime un nombre défini de messages dans un canal (réservé aux administrateurs).
6. !embed [Titre] + [Description] : Crée un embed stylisé et supprime le message original pour plus de clarté.
7. !info : Affiche des informations sur le bot : nombre de membres, latence et préfixe.
8. !invite : Fournit le lien d’invitation au serveur.
9. !perdu : Affiche la structure du serveur et aide les utilisateurs à s’orienter.
10. !ping : Vérifie la latence du bot.
11. !reglement : Affiche le règlement complet du serveur.

D/ MODERATION AUTOMATIQUE

- Suppression des messages contenant : -Mots interdits (liste configurable dans moderation.txt)
- Numéros de téléphone
- Adresses email
- Envoi d’un message d’avertissement au membre et création d’un log dans le canal de modération.


E/ NOTES IMPORTANTES
- Toutes commandes et fonctions du bot sont conçues pour respecter la vie privée des utilisateurs.
- Toutes les données traitées sont éphémères et limitées au temps d’exécution du bot.
- Le bot ne stocke aucune information personnelle et ne fait pas de profilage.

===== SENSIDYNIES BOT — USER GUIDE EN =====

SensiDynies is a Discord bot designed for French-speaking communities focused on chronic illnesses and well-being, aimed at providing a safe, respectful, and interactive environment.

A/ MAIN OBJECTIVES

- Security and moderation: Protect users by automatically removing inappropriate content, banned words, email addresses, or phone numbers.
- Information accessibility: Quick access to reliable medical definitions via !dico.
- Fun and interaction: Random jokes, random choices, and automatic reactions to make interactions more engaging.
- Server organization: Helps navigate and understand server categories with !perdu.
- Simplified communication: Easy access to bot information, server rules, and command guide.

B/ MAIN FEATURES

- Automatic detection of messages containing inappropriate language or personal information.
- Detailed logs in a moderation channel to track bot actions.
- Interactive commands to help members find information, have fun, or get oriented.
- Stylized embeds for better readability and visual appeal.
- Compatible with server custom emojis for automatic reactions.

C/ BOT COMMANDS

1. !aide [command] — Shows all commands or details for a specific command.
2. !dico [word] — Provides the definition of a medical term.
3. !blague — Posts a random joke for entertainment.
4. !choix [option1, option2, ...] — Randomly chooses one option from several.
5. !effacer [number] — Deletes a specified number of messages in a channel (admin only).
6. !embed [Title] + [Description] — Creates a stylized embed and deletes the original message.
7. !info — Shows bot information: number of members, latency, and prefix.
8. !invite — Provides the server invitation link.
9. !perdu — Displays server structure to help users navigate.
10. !ping — Checks the bot’s latency.
11. !reglement — Displays the full server rules.

D/ AUTOMATIC MODERATION

- Deletes messages containing: Banned words (configurable in moderation.txt)
- Phone numbers
- Email addresses
- Sends a warning message to the member and logs the action in the moderation channel.

E/ IMPORTANT NOTES

- All bot commands and functions are designed to respect users’ privacy.
- All processed data is temporary and limited to the command execution time.
- The bot does not store personal information and does not perform profiling.
